# spongemock_flutter

A Flutter project for translating text to [spongemock language](https://knowyourmeme.com/memes/mocking-spongebob).

## Building
### Build APK
```console
foo@bar: ~$ cd /path/to/project
foo@bar: ~$ flutter build apk
```

### Run
```console
foo@bar: ~$ cd /path/to/project
foo@bar: ~$ flutter run
```
